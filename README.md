# Projet Système Multi-Agents

## Description

Un projet de système multi-agents (SMA) développé dans le cadre du cursus M1 Informatique.

## Auteurs

- Hadjer CHEDJARI EL MEUR
- Etienne BOSSU

## Utilisation

```bash
# Lancer le projet (Arguments optionnels : nbRobots L1 L2 L3)
./run.sh 5 1000 2000 500  # Linux/Mac
run.bat  5 1000 2000 500  # Windows


# Générer la Javadoc
./doc.sh    # Linux/Mac
doc.bat     # Windows
```

## Structure du Projet

```
git_repo/
├── README.md      # Ce fichier
├── run.sh/bat     # Scripts d'exécution
├── doc.sh/bat     # Scripts de documentation
├── lib/           # JADE (jade.jar)
├── src/           # Sources Java 
└── rapport/       # Documentation LaTeX et PDF
```

## Prérequis

- Java
